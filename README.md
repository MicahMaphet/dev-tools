## bitcore-init
🚀 Quickly initalize everything I need for bitcore development using a desktop shortcut. Opens zed in bitcore, ghostty in bitcore-node, and brave.

### Settup
In the project directory:
```
chmod +x ./install.sh
```
```
./install.sh
```
Search for `bitcore-init` in apps and pin it to workspace

![bitcore-init on task bar](taskbar.png)

Now clicking the shortcut will start up my workspace
